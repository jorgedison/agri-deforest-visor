Leyenda NDVI

  • NDVI < 0.1 - Suelo expuesto
  • 0.1 - 0.2 - Vegetación muy degradada
  • 0.2 - 0.4 - Vegetación escasa o mixta
  • 0.4 - 0.6 - Vegetación moderada
  • 0.6 - 0.8 - Vegetación abundante
  • > 0.8 - Vegetación densa

Leyenda SAVI

  • SAVI < 0.2 - Suelo desnudo / Vegetación muy escasa

  • 0.4 - 0.6 - Vegetación moderada
  • 0.6 - 0.8 - Vegetación densa
  • > 0.8 - Vegetación muy densa

Leyenda NBR

  • NBR < -0.5 - Quemadura de alta severidad
  • -0.5 - 0 - Quemadura de severidad moderada
  • 0 - 0.2 - Áreas ligeramente quemadas
  • 0.2 - 0.4 - Vegetación estresada o en recuperación
  • 0.4 - 0.7 - Vegetación saludable
  • > 0.7 - Vegetación muy saludable
